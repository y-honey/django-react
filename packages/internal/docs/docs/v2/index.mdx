Welcome to the SaaS Boilerplate documentation! Our goal is to provide you with all the information you need to get up and running with the SaaS Boilerplate quickly and easily.

Whether you're just getting started with building your first SaaS product or you're a seasoned developer looking for a more efficient way to build high-performing and scalable applications, the SaaS Boilerplate has everything you need to succeed.

Our documentation is organized into five main sections:

- [Getting started](/v2/getting-started): This section contains instructions on how to set up and run a new project based on the SaaS Boilerplate, as well as how to run an existing instance of it.

- [Tutorials](/tutorials): Our tutorials section contains step-by-step guides on how to build different features using base code of the SaaS Boilerplate.

- [Working with SaaS Boilerplate](/working-with-sb): This section provides a detailed description of what's under the hood of the SaaS Boilerplate's features and best practices.

- [What is SaaS Boilerplate by Apptension](/v2/introduction): This section provides a high-level overview of the SaaS Boilerplate project and its architecture.

- [API Reference](/v2/api-reference): Our API reference section contains documentation on the commands used in the SaaS Boilerplate projects.
